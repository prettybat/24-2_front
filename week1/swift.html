Swift 문법 소개

swift란?

Swift는 Apple에서 개발한 프로그래밍 언어로, iOS, macOS, watchOS, tvOS 앱을 만들기 위해 주로 사용합니다
Swift의 주요 특징과 기본 문법을 간략히 소개하겠습니다

1. 기본 문법

변수와 상수

변수: var 키워드를 사용해 선언
상수: let 키워드를 사용해 선언

데이터 타입

int : 정수
double : 부동소수점
string : 문자열
bool : 불리언(true/false)

조건문

if문을 사용해서 작성

반복문

for-in과 while 사용

함수

func 키워드를 사용

배열과 딕셔너리

배열

같은 타입의 데이터를 순서대로 저장

딕셔너리

키-값 쌍으로 데이터를 저장

클래스와 구조체

객체지향프로그래밍을 지원
클래스와 구조체를 이용해 사용자 정의 데이터 타입을 생성 가능

옵션형

optional을 사용하여 값이 없을수도 있는 변수를 처리

결론

안정성과 간결함을 중시하는 현대적인 프로그래밍 패러다임을 수용한 언어
